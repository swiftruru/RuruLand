RuDjango 技術部落格平台

🔗 django.swift.moe

獨立開發並部署的全端部落格系統，採用現代化 Python Web 框架與雲端容器化部署架構。專案涵蓋完整的 CRUD 操作、使用者認證授權、即時通訊、安全性管控等企業級功能，實現從開發到生產環境的完整 DevOps 流程。

📋 核心功能與負責項目

  • 資料庫架構設計（PostgreSQL 關聯式資料表建模）
  • RESTful API 設計
  • 文章管理系統（WYSIWYG 編輯器整合）
  • 動態標籤分類系統（多對多關聯查詢優化）
  • 使用者身份驗證機制（Session 管理、密碼雜湊加密）
  • 持久化登入功能（Remember Me Token）
  • 個人資料管理模組
  • 密碼強度驗證與即時前端檢核
  • 技能標籤系統
  • WebSocket 即時私訊功能
  • IP 黑白名單安全管控
  • 登入嘗試記錄與異常偵測
  • Django Admin 後台擴充
  • CI/CD 自動化部署流程（Railway Platform）

🛠️ 技術棧（Tech Stack）

Python 3.x Django 6.0 Django ORM Django Channels PostgreSQL 17 Daphne ASGI HTML5 CSS3 JavaScript ES6+ Railway Docker Git
